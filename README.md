```https://github.com/hakil2001/plug_n_record_cam.git```

enter the dirctory by using this commad

```cd plug_n_record_cam```

```sudo chmod +x install.sh```

now, run the bash script 
```sudo bash install.sh  ```

```sudo ./install.sh```
